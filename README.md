# 🚇 Baku Metro Passenger Analysis

> Exploratory data analysis and machine learning on daily ridership data across Baku Metro stations — powered by open data from [data.gov.az](https://admin.opendata.az)

---

## 🗺️ About the Data

- **Source:** [opendata.az](https://admin.opendata.az/api/3/action/package_show?id=stansiyalar-uzre-gundelik-sernisin-gedislerinin-sayi)
- **Coverage:** Daily passenger counts per station (2025)
- **Scope:** 27 stations across 3 lines — 🔴 Red, 🟢 Green, 🟣 Purple

> Baku Metro first opened on **November 6, 1967** and today serves nearly **700,000 passengers daily**.

---

## 📊 Analysis Includes

| # | Analysis | Description |
|---|----------|-------------|
| 1 | 📍 Station Ranking | Total ridership per station |
| 2 | 📈 Monthly Trend | How ridership changes over time |
| 3 | 📅 Weekday Pattern | Busiest vs quietest days of the week |
| 4 | 🌡️ Heatmap | Station × Month ridership matrix |
| 5 | 🔍 Anomaly Detection | Unusual drops/spikes using Z-score |
| 6 | 🤖 Clustering | Grouping stations by behavior (KMeans) |
| 7 | 🔮 Forecasting | 30-day prediction using Prophet |
